PyCon APAC/Taiwan 是什麼？
2015 亞太 ／ 台灣 Python 年會 是為亞太及台灣區的 Python 愛好者舉辦的研討會。它將在 台北 舉行，而我們誠摯地邀請您加入這個盛大的活動，可透過贊助此活動來推廣您的品牌及商品，甚至透過它讓您的公司更加成功。前三屆的台灣 Python 年會（ 2012 , 2013 , 2014 ）是主要由下列的使用者社群來籌劃策辦：
  • Taipei.py - Taipei Python User Group
  • PyLadies Taiwan
  • Kaohsiung.py - Kaohsiung Python User Group
  • Django Girls Taipei
  • PyHUG - Python Hsinchu User Group
  • Tainan.py - Python Tainan User Group
  • Taichung.py - Taichung Python User Group
  • Hualien.py - Hualien Python User Group
